# TVHeadend EPG – Home Assistant

TVHeadend EPG integráció Home Assistant-hez HTTP kapcsolaton keresztül.

## Funkciók
- Külső TVHeadend szerver
- HTTP támogatás
- Kodi-szerű EPG megjelenítés (alap)
- HACS kompatibilis

## Telepítés
1. HACS → Custom repository
2. URL: https://github.com/yourname/tvheadend-epg-hacs
3. Típus: Integration
4. Újraindítás
5. Beállítások → Integrációk → TVHeadend EPG
